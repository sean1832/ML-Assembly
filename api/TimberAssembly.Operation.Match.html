    Methods

    Name Description
    CutToTarget(ref Remain)

    Cut the remainders to the target and create offcuts. (when target is smaller than subject)
    DoubleMatch(ref Remain)

    Two subjects from the remainders are combined to match one target. 1 dimensional matching.
    ExactMatch(ref Remain)

    One subject is exactly matched to one target.
    ExtendToTarget(ref Remain)

    Combining remainders with new subjects to match the targets. (when target is larger than target)
    RemainMatch(Remain)

    (DEPRECATED! Use ExtendToTarget Instead.)

    Match the rest of the targets with the rest of the subjects. Introduce offcuts if necessary.
    UniMatch(ref Remain)

    Four subjects from the remainders are combined to match one target regardless of orientations.

    WARNING: This method is slow! It is recommended to use after Double Match and Exact Match to reduce timber to match.
